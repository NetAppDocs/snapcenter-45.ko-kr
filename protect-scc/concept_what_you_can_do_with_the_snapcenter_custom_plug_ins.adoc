= SnapCenter 사용자 지정 플러그인으로 수행할 수 있는 작업
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
데이터 보호 작업에 SnapCenter 맞춤형 플러그인을 사용할 수 있습니다.

* 데이터베이스, 인스턴스, 문서 또는 테이블스페이스와 같은 리소스를 추가합니다.
* 백업을 생성합니다.
* 백업에서 복원합니다.
* 클론 백업.
* 백업 작업을 예약합니다.
* 백업, 복원 및 클론 작업을 모니터링합니다.
* 백업, 복원 및 클론 작업에 대한 보고서를 봅니다.
